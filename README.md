#Encriptación Simétrica con AES-CFB

Este programa permite al usuario **encriptar y desencriptar mensajes** utilizando el algoritmo **AES (Advanced Encryption Standard)** en modo **CFB (Cipher Feedback)**.

##Requisitos

- Python 3.x
- Librería [`pycryptodome`](https://pypi.org/project/pycryptodome/)

Instalación:
```bash
pip install pycryptodome


#Generador de Clave AES-128

Este script genera una clave de **128 bits (16 bytes)** utilizando la librería `pycryptodome` y la guarda en un archivo local para su uso posterior en algoritmos de encriptación simétrica como AES.

- Librería [`pycryptodome`](https://pypi.org/project/pycryptodome/)
